## Build & Deploy

**Development:**
```bash
npm run dev          # next dev --turbopack
npm run cf-typegen   # Regenerate cloudflare-env.d.ts
```

**Production:**
```bash
npm run deploy       # Clean -> opennextjs-cloudflare build -> deploy
npm run preview      # Local Cloudflare preview
npm run build        # Standard next build
```

**Scripts:**
```bash
npm run token:generate  # Generate API tokens (tsx scripts/manage-tokens.ts)
npm run token:list      # List API tokens
npm run token:revoke    # Revoke API tokens
```
