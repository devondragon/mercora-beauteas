```typescript
    id: string;

    stripe_customer_id: string;
    status: 'active' | 'paused' | 'canceled' | 'past_due' | 'incomplete' | 'trialing';
    current_period_start: string | null;

    // Joined data (parsed from JSON by model layer)
    plan_frequency: string;
    plan_discount_percent: number;
    product_name: string;       // Already parsed from JSON localized format
    product_slug: string;
    customer_name: string;      // Parsed from person JSON
    customer_email: string;     // Parsed from person JSON
    variant_price_amount: number; // In cents
  }>,
  stats: {
    activeCount: number;
    pausedCount: number;
    mrr: number;              // In dollars
    churnRate: number;        // Percentage 0-100
    newThisMonth: number;
    revenueTrendPercent: number; // Positive = growth, negative = decline
  },
  meta: { total: number; limit: number; offset: number }
}
```

GET /api/admin/subscriptions/:id
Response:
```typescript
{
  success: true,
  data: {
    subscription: {
      // Same fields as list item above for a single subscription
      id: string;
      customer_id: string;
      plan_id: string;
      stripe_subscription_id: string;
      status: string;
      current_period_start: string | null;
      current_period_end: string | null;
      created_at: string | null;
      plan_frequency: string;
      plan_discount_percent: number;
      product_name: string;
      product_slug: string;
      customer_name: string;
      customer_email: string;
      variant_price_amount: number;
    },
    events: Array<{
      id: string;
      subscription_id: string;
      event_type: 'created' | 'renewed' | 'payment_failed' | 'paused' | 'resumed' | 'skipped' | 'canceled' | 'updated';
      stripe_event_id: string | null;
      details: string | null;
      created_at: string | null;
    }>
  }
}
```

  <action>
Create a "use client" admin page following the exact patterns from `app/admin/orders/page.tsx`.

**State management:**
- loading, error states
- subscriptions array, stats object

- searchQuery string
- subsPerPage = 20

**Data fetching:**
- fetchSubscriptions(page, status, search) useCallback
  - `const offset = (page - 1) * subsPerPage`
  - `fetch(/api/admin/subscriptions?limit=${subsPerPage}&offset=${offset}&status=${status}&search=${search})`
  - Parse response, set subscriptions, stats, pagination state
- useEffect calls fetchSubscriptions on mount and when statusFilter or searchQuery changes (with debounce on search -- 300ms setTimeout)

**Stat cards section (SUBA-01):**
- 6 cards in a responsive grid: `grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 xl:grid-cols-6 gap-4`
- Each card uses `Card className="bg-neutral-800 border-neutral-700 p-6"` matching existing admin dashboard pattern
- Cards with icon, label (text-sm text-gray-400), value (text-2xl font-bold), subtitle (text-xs text-gray-500):
  1. Active Subscriptions: green color, Repeat icon, value = stats.activeCount
  2. Monthly Recurring Revenue: green color, DollarSign icon, value = `$${stats.mrr.toFixed(2)}`
  3. Churn Rate (30d): red/green conditional (>5% red, else green), TrendingDown icon, value = `${stats.churnRate.toFixed(1)}%`
  4. Paused: yellow color, Pause icon, value = stats.pausedCount
  5. New This Month: blue color, Plus icon, value = stats.newThisMonth
  6. Revenue Trend: green if positive/red if negative, TrendingUp/TrendingDown icon, value = `${stats.revenueTrendPercent > 0 ? '+' : ''}${stats.revenueTrendPercent.toFixed(1)}%`

**Filter bar:**
- Status tabs: row of buttons for All / Active / Paused / Canceled. Active tab uses `bg-orange-600 text-white`, others use `bg-neutral-800 text-gray-400 hover:bg-neutral-700`
- Search input: `Input` component with Search icon, placeholder "Search by customer name or email...", debounced onChange

  - Created: format created_at with `new Date(date).toLocaleDateString()`
  - Payment Status: Badge showing "Current" (green) for active, "Past Due" (red) for past_due, "N/A" (gray) for others

**Pagination controls:**
- Copy the exact pagination pattern from orders page: showing "X-Y of Z" text, First/Prev/numbered pages/Next/Last buttons
- Use ChevronsLeft, ChevronLeft, ChevronRight, ChevronsRight icons

**Loading state:**

**Error state:**
